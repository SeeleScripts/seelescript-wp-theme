# SeeleScript Starter WordPress Theme

This is a starter WordPress theme for [SeeleScript](https://seelescript.com).

## Author

**Carlos Hernandez**
- Website: [https://carlos.seelescript.com](https://carlos.seelescript.com)

## Features

- **Object-Oriented Programming (OOP):** The theme architecture is built using OOP principles for better organization and maintainability.
- **Tailwind CSS & daisyUI:** Utility-first styling with Tailwind CSS, complemented by daisyUI components.
- **_tw Based:** An npm project based on [_tw](https://underscoretw.com/), providing a modern development workflow.
